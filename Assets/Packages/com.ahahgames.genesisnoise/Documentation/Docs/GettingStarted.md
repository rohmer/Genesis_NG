In this directory, there are numerous examples on how to create various items using Genesis Noise.  Its a good place to take a look and try and grok what each example is doing

# Creating a new Genesis Noise Graph #
Everything Genesis Noise does, happens in a graph.  So once you are ready to create a new Graph:

1. In the Unity Project, right click on a folder
2. Genesis Noise->Noise Graph
3. Rename the noise graph to whatever you like
4. Double click on the new graph to begin editing
